# sequences/ — the derived running-read (novel)

One evolving entry per sequence: `SEQ NN - <name>.md`. **Derived, not authored** — a sequence entry is the running-read *of its scenes*. Authority flows up: when a scene below changes, `dev-capture` re-derives the sequence read; it never invents sequence content the scenes don't support.

A `GOAL → BUT → THEREFORE` chain spoken without a scene's concrete sensory grain routes here.
