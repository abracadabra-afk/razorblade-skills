# {{PROJECT}} — Contradiction Ledger

> **Silent capture, deferred resolution.** When a DEV discovery collides with **already-banked manuscript material** (a landed `draft.md`, a sealed thread, a paid-off plant), `dev-capture` logs a line here — **silently, never interrupting flow.**
> **Surface trigger (`surface_trigger` in frontmatter):**
> - `editing-seat` *(novel — default)* — surfaces only when CRE leaves the creative seat and asks, in the developmental editing pass, when the finished work can judge whether the change earned its disruption.
> - `ship-boundary` *(serial fiction)* — surfaces before the next installment publishes, because once shipped it can't be revised.
>
> Resolving the ledger is a **separate, future editing workflow** — out of scope for capture. This file only accrues.
