# dictation/

Raw transcripts go here. One file per dictation session.

**Naming:** `YYYY-MM-DD-NN-<segment-slug>.md` (NN = run number that day).

**Top of file:** pin the envelope segment this dictation was recorded against.

```
---
envelope-segment: 1
recorded: YYYY-MM-DD
---

<raw transcript>
```

These files are inputs to [[WORKFLOWS/transcoder|the Transcoder]]. Don't edit them — if a transcript is wrong, re-dictate.
